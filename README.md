# TypeScript FE Boilerplate
Boilerplate for an FE project using TypeScript, Bun Test Runner for testing and ESLint for linting
## Installation
- Install [bun](https://bun.com/docs/installation)
- Use `bun` to install all dependencies
```
bun install
```
## Usage
Use the `start` script to run the project for development
```
bun run start
```
To generate the distributables, run the `build` script
```
bun run build
```
